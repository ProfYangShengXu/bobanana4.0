🎴 Reasonix Cyclic Workflow Plugin

版本: 1.0.0 · 三阶段: 设计 → 编码 → 验证

安装

  1. 把 bin/ 目录复制到你项目的 .reasonix/../ 或放到 PATH 中
  2. 在项目根目录运行 install.bat（或手动复制 skills/ 和 commands/ 到 .reasonix/ 下）
  3. 在 reasonix.toml 添加插件配置

文件清单

dist/cycle-plugin/
├── bin/
│   ├── cycle-bridge.exe    ← MCP 插件（状态管理：queue_next_prompt / signal_done）
│   └── reasonix.exe         ← 主程序（含 cycle 子命令）
├── skills/
│   ├── cyclic-workflow/
│   │   └── SKILL.md         ← 三阶段元指令（17KB）
│   └── cycle-install/
│       └── SKILL.md         ← 安装器 skill（2KB）
├── commands/
│   └── cycle.md              ← /cycle 斜杠命令（渲染为 run_skill 调用）
├── install.bat               ← 一键安装脚本
└── README.html               ← 本文件

使用

  • TUI / 桌面端: 输入 /cycle 你的目标
  • 终端: reasonix cycle "你的目标"
  • 全局安装: /skill cycle-install（在任意 workspace 的 TUI 中）

构建

go build -o bin/reasonix.exe  ./cmd/reasonix/
go build -o bin/cycle-bridge.exe ./cmd/cycle-bridge/

构建依赖：Go 1.25+
